# Brainstorm

This folder is for exploratory notes and proposals. Content is informal, but still English by default.

## Folders
- `docs/brainstorm/inbox/`: quick dumps, unsorted notes
- `docs/brainstorm/concepts/`: pillars, core loop, themes
- `docs/brainstorm/mechanics/`: movement, abilities, hazards, combat (if any)
- `docs/brainstorm/tech/`: engine/tooling/pipeline ideas
- `docs/brainstorm/levels/`: level patterns, setpieces, traversal beats
- `docs/brainstorm/ui-ux/`: HUD/menus/onboarding/accessibility
- `docs/brainstorm/templates/`: templates for consistent notes
- `docs/brainstorm/archive/`: obsolete notes

## Template
Start with `docs/brainstorm/templates/proposal.md` for anything that could become a decision.
