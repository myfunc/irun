# IRUN

IRUN is an early-stage **3D platformer** written in **Python**.

## Quick Start

Requirements:
- Python 3.11+ recommended

Install and run:
```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -e ".[dev]"
python -m irun
```

Smoke run (starts and exits quickly):
```bash
python -m irun --smoke
```
