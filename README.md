# IRUN

IRUN is a monorepo for multiple applications related to the project.

## Apps
- `apps/up`: UP, the main 3D platformer game (Python + Ursina)

## Quick Start (UP)
```bash
cd apps/up
python -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
python -m pip install -e ".[dev]"
python -m up
```
